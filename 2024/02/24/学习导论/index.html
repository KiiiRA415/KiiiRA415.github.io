# 学习导论

学习的本质是突触的形成和重构，研究如何学习，如何提高学习效率，都应该从如何优化突触的形成和重构入手

# 既定范围内的信息以某种算法多次重现

这个理论总结于我在查留学资料的过程中，当时，我注意到我对于同一个概念社区大学参考过视频，论坛的多个帖子，多个网页，这其中的原理是，我框定一个范围社区大学，然后以从最优顺序的主动搜索为算法，多次重复这个概念

相同的例子还有背单词，我们默认的背单词方法是框定单词本身和含义的这一极小的范围，然后以记忆曲线为算法多次重复，但事实上，肯定有更有的办法，比如优化范围本身，不吝啬于单词的出现地，单词可以出现在句子里，视频里，网页里，还可以优化算法，不限定于记忆曲线，可以按心情自己重复，可以按睡眠时间重复，可以用土方子：便利贴贴冰箱，这样就利用出入厨房构成了算法，而出入厨房的行为本身决定了重复次数的保障

高三冲刺的过程中有的同学进步很快，这是因为高三的卷子把题目范围限定在了高考题型内，然后由学校研究高考的专家安排算法，考试频率变高，而又增加数量，所以使得基础好的同学很快就能提分。在这过程中，我必须提醒的是，我对于高考的基础的定义是你是否会做高考题的简单中档部分

# 任何安排都应服务于方便睡觉

睡觉本身就是突触自我形成和修复的过程，如果说白天进行了输入，那么是否有充足的睡眠就决定了是否真的按下了 “保存键”, 熬夜是极其有害的行为，不仅伤身体，而且会使得白天学过的东西发生损耗

提升学习的效率本质是加快突触的形成和重构，而任何经过检验的学习方法都不应该与睡眠冲突，如果有，应当舍去

任何学习安排本身应该服务于睡觉，我不阻止你一上午暴学，但你必须在午睡之前停下来，而且要保证午睡时间

# 不吝啬于大量产出

产出本身不一定能带来大量好处，产出的信息本身不一定对谁有好处，但产出本身一定不会有害处，你在学过微积分之后，你可以写一份微积分的总结，画一幅微积分的总结图，可以教别人算积分，可以发表一份微积分的教程到自己博客上，虽然这个教程可能写的晦涩难懂，只能教你自己，没有流量，但是他没有害，反而，在写教程的过程中你自己重新梳理了一遍微积分的内容，这对你是有益的

因此，产出这个行为本身是无害的，他具体能产生多大好处取决于你自己

既然产出的行为本身无害，那么在产出前进行的就不是好歹的决策，而是孰优孰劣的决策，而这个优劣又是相对的，取决于许多因素

例如，如果考试将至，虽然刷错题和刷新题都是产出行为的子集，本身各有作用，没有所谓的孰优孰劣，但考虑到 “如何提高考试的容错率” 的问题，那么选择刷错题就是更优的决策；光讨论 CS 专业课的学习，那么刷线上题库，做项目，在论坛上跟码友讨论，写个人博客… 都是能想到的产出行为，但考虑到作为毕业生面试的个人名片，对读研读博导师的个人宣传，考虑到学习内容的共享性和易于传播性，再考虑到产出内容的商用等… 这个时候选择个人博客来产出就是最好的选择，举一反三，你也应该和你用网站管理文章一样，用代码托管网站管理代码，收藏夹里的网站不要只是自己用，也应该整理成 markdown 文章方便分享
