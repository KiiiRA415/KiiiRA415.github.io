# 理论一

学习语言的五个步骤

  1. 打好基础

    ①学基本的字母发音

    ②积累基本的单词 (1000~2000 个)

    ③积累基本的语法

  2. 搜集按难度梯度划分的影片

    先开字幕预估难度，控制在 50%~60% 都会的程度最好

  3. 开着字幕看影片

  4. 随着影片量的提升逐渐加大难度

    每天最少 1h, 一般 1~2h, 想快速进步就 3h+

  5. 开始练习口语

    写日记，自言自语

PS: 不建议在 1. 上花太多时间，应该把大量时间用在 2.~4. 上

# 理论二

几种背单词的情境

  1. 长时间期限慢速积累

    例如托福单词，我从高二开始背，实际上大二才需要用，我就直接交给墨墨背单词了，一天 10~30 个

  2. 短期巩固大段单词

    是打基础的一环，例如日语从零单词量到 1000~2000 单词量，直接用词典软件选中小学初中词汇，然后快速看单词，每个单词停留 10s 以内，堆次数

  3. 自然习得

    在冲浪，看影片的时候遇见不会的单词，用划词插件看意思，然后放过 (不想放过就加入生词，不过没什么必要，从来是增加负担，因为这有点打乱 CI 理论的节奏)

# 理论四

良好的语言学习观念

  1. 必须保证每天的有质有量的练习，坚持下去
  2. 语言学习必然会经过快速成长期，停滞期，稳步拔高期，要接受时间投入与产出的不均衡
  3. 心想 “我要成为什么样的人”, 而不是 “我要完成什么”

# 理论五

养成习惯的方法

  1. 让提示变得明显

    ①给油管订阅分类

    ②把每天要做的事具体写出，格式类似于 “我在什么时候在哪做什么事”

    ③与已经有的习惯相配合

    ④直接能看到要做的事，比如提前打开电脑，打开油管

  2. 三思而后行

    几秒钟的选择就会决定之后几个小时的干的事，就跟 gal 的选项一样，在做什么事之前要三思，你的任何一次决策都是在给 “我要成为什么样的人” 的投票

反馈机制
